# 🎂 25th Birthday Surprise Website

A beautiful dark-themed birthday website for your brother, built with HTML, CSS, and JavaScript.

## How to Use

### 1. Add Your Photos
Replace the placeholder images in the `images/` folder with your own photos:

| Filename | Recommended Size | Where it appears |
|----------|-----------------|------------------|
| `photo-main.jpg` | 600x700 | About section - main photo |
| `photo-1.jpg` | 800x600 | Gallery - large feature image |
| `photo-2.jpg` | 600x600 | Gallery |
| `photo-3.jpg` | 600x600 | Gallery |
| `photo-4.jpg` | 800x400 | Gallery - wide image |
| `photo-5.jpg` | 600x600 | Gallery |
| `photo-6.jpg` | 600x600 | Gallery |

> **Tip:** If you don't add photos, beautiful default images will show automatically. But personal photos make it truly special!

### 2. Customize the Text
Open `index.html` and edit the text sections:
- Hero section message
- About section description
- Timeline events (add real milestones!)
- Birthday wishes and messages
- The special message at the bottom

### 3. Preview the Website
Simply open `index.html` in any web browser:
```bash
open index.html
```

Or if you want to see it on your phone too, you can use a simple server:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`

## Features

✨ **Dark theme** with blue-green accents  
✨ **Animated particle background** in hero  
✨ **Scroll animations** throughout  
✨ **Interactive photo gallery** with lightbox  
✨ **Animated timeline** of life journey  
✨ **Floating cards** and glassmorphism effects  
✨ **Confetti surprise** when scrolling to wishes
